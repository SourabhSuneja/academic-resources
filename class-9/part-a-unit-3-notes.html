ICT Skills - Notes

1. What is ICT?

ICT stands for Information and Communication Technology. It refers to all the technology used to handle telecommunications, broadcast media, intelligent building management systems, network-based control systems, and processing information.

2. What is a Computer?

A computer is an electronic device that accepts raw data as Input, processes it according to a set of instructions (program), and produces the result as Output.

3. The IPO Cycle

Computers work on the principle of the Input-Process-Output (IPO) cycle:

  • Input: Entering data and instructions (e.g., via Keyboard).
  • Process: Working on the data (done by the CPU).
  • Output: Giving the result (e.g., on Monitor).

4. The CPU (Processing Device)

[Image of CPU Diagram]

The Central Processing Unit (CPU) is called the "Brain of the Computer." It consists of three main parts:

  • ALU (Arithmetic Logic Unit): Performs mathematical calculations (+, -, *, /) and logical comparisons (<, >, =).
  • CU (Control Unit): Controls the flow of data and guides the computer how to execute instructions. It acts like a traffic policeman.
  • MU (Memory Unit): Temporarily stores data and instructions during processing.

5. Computer Memory

Memory is where a computer stores data. It is classified into two types:

  • Primary Memory: The main memory directly accessible by the CPU.
    - RAM (Random Access Memory): Volatile (temporary) memory.
    - ROM (Read Only Memory): Non-volatile (permanent) memory.
  • Secondary Memory: Used for permanent storage (e.g., Hard Disk, SSD, Pen Drive, CD/DVD).

6. Memory Measuring Units

Data in a computer is stored in the form of 0s and 1s (Binary Digits).

7. Input and Output Devices

Category Function Examples
Input Devices Send data to the computer Keyboard, Mouse, Scanner, Microphone, Joystick, Webcam, Barcode Reader
Output Devices Receive data from the computer Monitor (VDU), Printer, Speakers, Projector, Plotter, Headphones

8. Hardware Components

  • Motherboard: The main circuit board that connects the CPU, memory, hard drives, and ports together.
  • Ports: Slots on the computer to connect external devices.
    - USB: For Pen drives, Mouse.
    - HDMI/VGA: For Monitors/Projectors.
    - Audio Port: For Speakers/Mic.

9. Operating System (OS)

An Operating System is a system software that acts as an interface between the user and the computer hardware.

Functions of OS:

  • Process Management: Decides which task gets the CPU.
  • Memory Management: Allocates memory to programs.
  • File Management: Organizes files and folders.
  • Device Management: Controls input/output devices.

10. Types of Operating Systems

  • Single User: Supports one user at a time (e.g., MS-DOS).
  • Multi-User: Allows multiple users simultaneously (e.g., UNIX, Linux).
  • Real-Time: Used in robotics and critical systems requiring immediate response.

11. Windows 10 and Operations

Windows 10 is a popular GUI (Graphical User Interface) based OS developed by Microsoft.

  • Features: Start Menu, Cortana (Voice Assistant), Task View, Action Center.
  • File Management: Creating folders (Right-click > New > Folder), Copying (Ctrl+C), Moving (Ctrl+X), Pasting (Ctrl+V).
  • Customization: Changing Wallpapers and Screen Savers via Personalization settings.

12. Booting

Booting is the process of starting the computer and loading the OS.

  • Cold Booting: Turning on a computer from a completely off state.
  • Warm Booting: Restarting a computer that is already on (using Restart or Ctrl+Alt+Del).

13. The Internet

The Internet is a global network of interconnected computers (Network of Networks). History: It started in 1969 as ARPANET by the US Department of Defense.

How it works: It follows the Client-Server model. The Client (your PC) sends a request, and the Server provides the information.

Applications: E-learning, E-commerce (Shopping), Social Networking, Online Banking, Entertainment.

14. Common Internet Terms

  • WWW (World Wide Web): Collection of websites accessible via the internet.
  • ISP (Internet Service Provider): Companies that provide internet access (e.g., Jio, Airtel).
  • URL (Uniform Resource Locator): The address of a website.
  • Web Browser: Software to access the web (e.g., Chrome, Edge).

15. Internet Protocols

Protocols are rules that govern data communication.

  • TCP/IP: Basic communication language of the internet.
  • HTTP: HyperText Transfer Protocol (used for viewing webpages).
  • FTP: File Transfer Protocol (used for uploading/downloading files).

16. Email (Electronic Mail)

Email is used to exchange messages digitally using the internet. Important fields when composing:

  • To: Email address of the main recipient.
  • CC (Carbon Copy): Sends a copy to others; everyone can see who received it.
  • BCC (Blind Carbon Copy): Sends a copy, but other recipients cannot see this person's email.
  • Subject: A short summary of the email content.

17. Digital India

Digital India is a campaign launched by the Government of India to ensure that government services are made available to citizens electronically by improving online infrastructure and Internet connectivity.
