ICT Skills - Q&A

Short Answer Type Questions

1. Define ICT.

ICT stands for Information and Communication Technology. It refers to all the tools and technologies used to handle, store, and communicate information electronically.

2. Which ICT skills do you need at home?

At home, we need skills to operate smartphones, smart TVs, browse the internet for information, use social media, and perform online banking.

3. What is a light pen?

A light pen is a light-sensitive pointing input device, shaped like a pen, used to select a displayed menu item or draw pictures on the monitor screen.

4. What are laser printers? Where are they used?

Laser printers are high-quality non-impact printers that use laser beams to print. They are mostly used in offices and publishing houses for fast and sharp printing.

5. What does Motherboard consist of?

A motherboard consists of the CPU, RAM, ROM, BIOS, expansion slots, and various ports to connect input/output devices.

6. Give two points of difference between email and postal mail.

Email is instant and usually free (requires internet), whereas postal mail takes days to deliver and costs money for envelopes and stamps.

7. What are protocols? Give two examples.

Protocols are a set of rules that determine how data is transmitted over a network. Examples are HTTP and TCP/IP.

8. What is a web browser? Give an example.

A web browser is an application software used to access and view websites on the internet. Example: Google Chrome.

Long Answer Type Questions

1. What are the impacts of ICT in our professional life?

ICT has transformed professional life by:

  • Increasing speed and accuracy of work.
  • Enabling communication through email and video conferencing.
  • Allowing remote work (Work from Home).
  • Automating repetitive tasks.

2. Name all the components of ICT with example.

The main components of ICT are:

  • Hardware: Physical parts like Computer, Mobile.
  • Software: Programs like Windows, MS Word.
  • Data: Raw facts and figures.
  • People: Users who operate the system.
  • Procedure: Rules to follow.

3. Why is the motherboard an important component of a computer?

The motherboard is crucial because it acts as the backbone of the computer. It connects the CPU, memory, hard drives, and all peripherals (mouse, keyboard) together, allowing them to communicate with each other.

4. Explain the IPO cycle with an example.

Computers work on the Input-Process-Output cycle.

  • Input: Taking data in (e.g., Typing 2 + 2).
  • Process: Calculating data (e.g., CPU calculates 4).
  • Output: Showing result (e.g., Monitor shows 4).

5. What is the difference between input and output devices?

  • Input Devices: Used to send data into the computer. Example: Keyboard, Mouse, Scanner.
  • Output Devices: Used to receive data from the computer. Example: Monitor, Printer, Speaker.

6. Differentiate between Windows and Linux operating systems.

  • Windows: It is a paid OS owned by Microsoft, very user-friendly, and has huge software support.
  • Linux: It is a free and open-source OS, highly secure, and mostly used by developers and servers.

7. Mention important features of UNIX.

UNIX is a powerful operating system known for being:

  • Multi-user: Many users can use it at once.
  • Multi-tasking: Can run multiple programs simultaneously.
  • Portable: Can work on different types of hardware.

8. Write down steps to shut down a computer.

To shut down a Windows computer safely:

  • Click on the Start button in the bottom-left corner.
  • Click on the Power icon.
  • Select the Shut down option from the menu.
