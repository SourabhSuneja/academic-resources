Data Visualizations - Q&A

Short Answer Type Questions

1. What is the importance of data visualization techniques?

Data visualization helps us understand large and complex data easily by showing it in the form of charts, graphs, and diagrams.

2. Why is trend analysis given more importance?

Trend analysis helps in identifying patterns and changes over time, which is useful for predictions and decision-making.

3. In which case, is it advisable to go for dot plot?

A dot plot is useful when we want to show small sets of numerical data and compare individual values clearly.

4. Does a bar chart or bar graph present categorical data?

Yes, a bar chart is used to represent categorical data by comparing different categories.

5. What is the difference between range and frequency?

Range shows the difference between the highest and lowest values, while frequency shows how often a value occurs.
